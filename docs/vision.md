# Vision

Build a lightweight, local-first compliance authoring tool.

The application allows users to document security control implementations and export valid OSCAL artifacts.

The first supported baseline for the MVP is a small subset of NIST SP 800-53 Rev. 5 Moderate.

Future work may add FedRAMP-specific policy evaluation (Consolidated Rules) and, if an official FedRAMP OSCAL profile is located and approved, FedRAMP baseline support. The application does not currently implement an official FedRAMP baseline.

The application is not intended to become a full GRC platform.
